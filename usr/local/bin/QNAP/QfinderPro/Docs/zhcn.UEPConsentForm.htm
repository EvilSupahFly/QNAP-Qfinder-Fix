感谢您使用威联通科技股份有限公司 QNAP System Inc.（下称“本公司”）产品。为提供良好的用户体验，当您使用 QTS（下称“本产品”）时本公司将收集相关使用信息，详细说明如下：

目的

用户信息及用户行为可以帮助本公司更好地了解用户习惯及偏好。本公司收集此类信息来改进产品与服务，使之更加契合用户需求，提升整体服务质量。

收集的信息

收集的信息包含：

1.  操作系统信息、设备识别码、国家或地区及语言设置、计算机型号、固件版本等基本信息。

2.  应用程序相关信息，包含：版本信息、更新与关闭时间、使用频率和使用时间。

3.  用户偏好：设备设置、产品配置、应用程序与硬件的使用时间。

4.  其他相关但不涉及个人隐私的信息。

信息的使用

您同意后，前文所述的信息将被自动收集并发回本公司。本公司将对收集的信息进行分析，找出可对未来产品和服务进行的改进。本公司具有有效机制与程序来保障所收集信息的安全性，并仅将相关信息内部使用。
