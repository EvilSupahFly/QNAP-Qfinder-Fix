Συγκατάθεση για συλλογή πληροφοριών χρήστη

Σας ευχαριστούμε που χρησιμοποιείτε τα προϊόντα της QNAP Systems, Inc. (εφεξής καλούμενη ως η "Εταιρεία"). Για να σας παρέχουμε καλύτερη εξυπηρέτηση χρήστη, η Εταιρεία θα συλλέγει σχετικές με τη χρήση πληροφορίες όταν χρησιμοποιείτε το QTS (εφεξής καλούμενο ως το "Προϊόν"), όπως αναφέρεται λεπτομερώς παρακάτω:

Σκοπός

Οι πληροφορίες χρήστη και η συμπεριφορά χρήστη βοηθούν την Εταιρεία να κατανοήσει καλύτερα τις συνήθειες και τις προτιμήσεις του χρήστη. Η Εταιρεία συλλέγει τις πληροφορίες αυτές για να βελτιωθούν το Προϊόν και οι υπηρεσίες ώστε να ικανοποιούνται οι ανάγκες των χρηστών και να βελτιωθεί η συνολική ποιότητα υπηρεσιών.

Πληροφορίες που συλλέγονται

Στις πληροφορίες που συλλέγονται περιλαμβάνονται:

1.      Πληροφορίες του λειτουργικού συστήματος, κωδικοί αναγνώρισης συσκευής, ρυθμίσεις χώρας και γλώσσας, μοντέλο υπολογιστή, έκδοση υλικολογισμικού και άλλες βασικές πληροφορίες.

2.      Πληροφορίες σχετικές με την εφαρμογή συμπεριλαμβανομένων των εξής: πληροφορίες έκδοσης, ώρα ενημέρωσης και τερματισμού, συχνότητα χρήσης και χρόνος χρήσης.

3.      Προτιμήσεις χρήστη: ρυθμίσεις συσκευής, διαμόρφωση προϊόντος, χρόνος χρήσης της εφαρμογής και υλισμικό.

4.      Λοιπές σχετικές αλλά μη προσωπικές πληροφορίες.

Χρήση πληροφοριών

Με τη συμφωνία σας, οι προαναφερθείσες πληροφορίες θα συλλεχθούν αυτόματα και θα αποσταλούν ξανά στην Εταιρεία. Η Εταιρεία θα αναλύει της συλλεχθείσες πληροφορίες για να προσδιοριστούν οι βελτιώσεις που μπορούν να γίνουν σε μελλοντικά προϊόντα και υπηρεσίες. Η Εταιρεία έχει αποτελεσματικούς μηχανισμούς και διαδικασίες για την προστασία στης ασφάλειας των πληροφοριών που έχουν συλλεχθεί και θα χρησιμοποιεί μόνον εσωτερικά τις σχετικές πληροφορίες.

Αποποίηση ευθύνης

Με το παρόν, η Εταιρεία αποποιείται όλων των εγγυήσεων συμπεριλαμβανομένων ρητών ή σιωπηρών εγγυήσεων εμπορευσιμότητας ή καταλληλότητας για κάποιο συγκεκριμένο σκοποί σε σχέση με αυτή τη συγκατάθεση ή με οποιονδήποτε τρόπο. Πέρα από τα προαναφερθέντα, η Εταιρεία δεν θα θεωρείται υπόλογη για κάποια άμεση, έμμεση, ειδική, αποθετική ή συμπτωματική ζημία, όπως απώλεια εσόδων, απώλεια δεδομένων, χρήση εξοπλισμού ή λειτουργική ζημία, διακοπή επιχειρηματικών δραστηριοτήτων και συμβάντων ή παρόμοιας φύσης, ανεξαρτήτως του αν υπάρχει προηγούμενη ειδοποίηση για αυτό το συμβάν ή όχι.
